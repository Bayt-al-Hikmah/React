# React Fundamentals Course

Welcome to the **React Fundamentals Course** repository by Bayt-al-Hikmah!  
This collection of workshop-based lecture notes is designed for beginners and intermediate learners who want to master the core principles of React and modern front-end development. Whether you're transitioning from basic web development or strengthening your existing skills, this course provides a structured, progressive path to understanding React from the ground up.

Bayt-al-Hikmah (House of Wisdom) is dedicated to preserving and spreading knowledge, inspired by the flourishing intellectual spirit of the Islamic Golden Age. These workshops emphasize clarity, practicality, and hands-on learning to help you build real, scalable front-end applications using React.

### Setup Instructions

1. **Install Node.js & npm**  
    Download from nodejs.org and verify installation using:
    
    `node -v npm -v`
    
2. **Install VS Code**  
    Add the recommended extensions:
    
    - ES7+ React/Redux Snippets
        
    - Prettier (for formatting)
        
3. **Create Your First React App**  
    You can bootstrap using:
    
    `npx create-react-app my-app`
    
4. Run your development server:
    
    `npm start`

## Course Overview

This course is organized into **4 progressive workshops**, each focusing on fundamental concepts through theory, examples, and practical activities.

|**Workshop**|**Topic**|**Key Concepts Covered**|
|---|---|---|
|Workshop 1|Intro to React & SPAs|Single Page Applications (SPA), React core concepts, components, JSX, creating and understanding a React App.|
|Workshop 2|State & Rendering|`useReducer`, children components, rendering lists with `.map()`, conditional rendering, controlled components & handling user input.|
|Workshop 3|Side Effects & Routing|Managing side effects with `useEffect`, client-side routing using React Router, Context API for global state, accessing the DOM using `useRef`.|
|Workshop 4|Advanced Concepts|Default props & prop validation, custom hooks, memoization, code splitting & lazy loading, Higher-Order Components (HOCs), portals.|

## Practice and Projects

- **Exercises**: Each workshop includes coding challenges and guided activities.
    
- **Mini-Project Ideas**:
    
    - **Workshop 1**: Build a simple SPA with multiple components.

    - **Workshop 3**: Build a small multi-page app with routing and global state.
        
    - **Workshop 4**: Refactor your app using custom hooks, lazy loading, and memoization.
